<h1>人工智能助手</h1>
<p>适用于边缘侧的人工智能应用，通过网页进行操作，简单易用，便捷高效</p>
<h2>功能概要</h2>
<ul>
<li>调用边缘侧或远程摄像头，将画面投送到浏览器页面上，并进行人脸检测</li>
<li>控制服务器远程关机，启停摄像头</li>
</ul>
